
# Maxgram

A clone of Instagram created with MERN stack.

## Demo

https://maxgram.zabarka.com/
## Features

- Authentication
- Creating / deleting posts
- Comments
- Saving posts
- Liking posts
- Following users
- Customizing profile
- Multiple images per post
- Explore page
- User search
## Tech Stack

- MongoDB
- Express
- React
- Node
- SCSS
- Multer
- AWS
- pm2
